**How to start?

$ git clone https://github.com/ayushisingla/Frequency-Counter.git

$ npm init

$ npm install express --save

$ npm install nodemon

$ npm i node-fetch

$ nodemon index.js





**What used?

1. NodeJS

2. Express framework

3. Nodemon ( for ease, not definitely required)

4. body parser



**Components are explained in the files itself.
